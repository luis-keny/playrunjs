## üöÄ Usage  
1. Write code in the editor.
2. Results will appear in the console on the right.  

**Example:**  
```javascript  
// Calculate Fibonacci  
function fibonacci(n) {  
  return n <= 1 ? n : fibonacci(n - 1) + fibonacci(n - 2);  
}  

console.log(fibonacci(10)); // 55  
```
